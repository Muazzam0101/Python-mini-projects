Quiz Game Description
Welcome to the Quiz Game! This interactive Python program is designed to challenge your knowledge on various topics through a series of multiple-choice questions. Whether you're a trivia enthusiast or just looking for some fun, this game offers an engaging experience with increasing levels of difficulty.

Gameplay Overview
Question Structure:

The game consists of 10 questions, each with four possible answer options labeled A, B, C, and D.
The questions cover a diverse range of topics, including science, literature, geography, history, and more.
Answering Questions:

For each question, you must choose the correct answer by inputting the corresponding letter (A, B, C, or D).
The game provides instant feedback on whether your answer is correct or not.
Winning Prizes:

With each correct answer, you accumulate winnings. The prize amounts increase progressively, creating a sense of achievement as you advance through the questions.
The total winnings are displayed after each correct answer.
End of Game:

If you provide an incorrect answer, the game ends, and your total winnings are revealed.
The total winnings are determined based on the last correctly answered question.
Included Files
quiz_game.py:
The heart of the game, this Python script contains the logic for presenting questions, accepting user input, checking answers, and managing prize winnings.
How to Run
Prerequisites:

Ensure you have Python installed on your system.
Execution:

Open a terminal or command prompt.
Navigate to the directory containing quiz_game.py.
Run the script with the command: python quiz_game.py.
Customization
Feel free to modify the script to add more questions, change prize amounts, or adjust any other aspects of the game. You can tailor the quiz to suit specific themes or subjects that interest you and your audience.

Enjoy the Challenge!
